About the Track “Square of Numbers” Task: transform the input CSV so that in the output the column num equals the square of the original value while preserving the mapping by id. Format: input input.csv with header id,num; output output.csv with the same header; keep the row order. Integers and floats are allowed. Execution: the organizer runs python3 main.py input.csv; your script must create output.csv in the current directory. Scoring: number of correctly computed rows plus a bonus from random.random() as a tie-breaker.
