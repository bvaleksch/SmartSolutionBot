Track Rules: “Square of Numbers” 1. Submission: a single ZIP archive with main.py at the root. 2. Execution: python3 main.py input.csv; the script must create output.csv in the current directory. 3. Input: CSV (UTF-8, comma) with header id,num. 4. Output: CSV with header id,num, where num = square of the input; preserve row order. 5. Scoring: count of correct rows + bonus random.random() as a tie-breaker. 6. Error handling: non-numeric num values are not counted; do not crash on empty lines or whitespace. 7. File names: strictly input.csv and output.csv.
